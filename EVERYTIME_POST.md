에타 친구들 안녕! 👋
매번 학기 시작할 때마다 시간표 짜느라 엑셀 두드리고, 머리 쥐어짜면서 경우의 수 계산하기 너무 귀찮았지?

시간표 짤 때 머리 싸매지 말라고 자동으로 겹치지 않는 시간표 조합을 싹 다 계산해서 뽑아주는 웹 서비스를 만들었어!

완전 무료고, 광고도 없고, 개인정보도 수집 안 하니까 편하게 와서 써봐! 🚀

✨ 주요 기능 요약
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 📝 복사-붙여넣기 자동 인식 (수강요람 파서)
- 학교 수강요람이나 에타 과목 텍스트를 그냥 쭉 긁어서 복사 붙여넣기만 하면 과목명, 교수명, 시간표 정보가 자동으로 파싱되어 입력돼! (직접 하나씩 안 적어도 됨!)

2. 🔀 모든 가능한 시간표 조합 자동 생성
- 과목들을 다 선택하고 [시간표 생성] 버튼만 누르면, 겹치지 않는 모든 시간표 경우의 수가 비주얼 카드로 정렬되어 쫙 쏟아져 나와!

3. 🔍 강력한 맞춤형 필터
- 우주공강 극혐하는 사람들을 위한 [연강 우선 필터]
- 아침 잠 많은 사람들을 위한 [1교시(9시) 제외 필터]
- 황금 금요일을 위한 [금공강 필터]
- 싫어하는 교수님 수업은 알아서 걸러주는 [교수님 필터]

4. 🎨 트렌디한 다크 모드 & 고대비 UI
- 밤에 에타 보면서 시간표 짜도 눈 안 아프게 감성적인 다크 모드 테마를 완벽 지원해!

💡 사용 방법 (초간단 3초 컷)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

[1단계] 수강요람이나 에타 과목 텍스트를 긁어서 입력창에 붙여넣는다.
[2단계] 듣고 싶은 과목들을 선택하고 하단의 [시간표 생성]을 클릭한다.
[3단계] 필터 탭에서 나만의 조건(금공강, 아침 제외 등)을 켜며 마음에 드는 시간표를 고른다!
[4단계] 결정한 시간표는 이미지나 JSON 파일로 간편하게 백업/저장한다!
